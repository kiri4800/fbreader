Class Overview

The class that saves/restores book info in the app. There is no default implementation for most methods of the class. This is the app developer responsibility to implement the methods. You can extend this class directly or use AbstractTextBookController class that provides empty implementation for all methods as the base class.

public abstract void iterateVisibleBookmarks (TextBookController.BookmarksConsumer consumer)

Iterates visible bookmarks. There is no method returning list of all wisible bookmarks because if book has thousands bookmarks, such method can take a long time. It is recommended to pass small chunks to the consumer to allow the view to call UI update after processing of each chunk.
